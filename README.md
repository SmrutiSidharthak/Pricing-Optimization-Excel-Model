# Pricing-Optimization-Excel-Model
Pricing Optimization &amp; Profit Simulation Model using Excel and Solver

## Overview
This project focuses on determining the optimal pricing strategy to maximize profit using Excel-based modeling and optimization techniques.

## Problem Statement
Businesses often struggle to balance pricing, demand, and profitability. Setting prices too high reduces demand, while low prices reduce margins.

## Objective
- Build a dynamic pricing model
- Analyze the relationship between price and demand
- Identify the profit-maximizing price
- Evaluate the impact of demand sensitivity

## Methodology
- Developed a demand model based on price sensitivity
- Calculated profit using contribution margin approach
- Performed sensitivity analysis (Price vs Profit)
- Used Solver to identify optimal pricing

## Key Result
- Optimal Price: ₹141.43
- Maximum Profit: ₹52,014

## Key Insights
- Profit depends on the trade-off between margin and demand
- Higher prices increase margin but reduce demand
- Optimal pricing balances both factors
- Demand sensitivity significantly impacts pricing decisions
